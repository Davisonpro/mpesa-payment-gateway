# WordPress.org Assets

This directory contains assets for the WordPress.org plugin directory.

## Required Files

### Screenshots (required for plugin listing)

Add the following screenshots in this directory:

1. `screenshot-1.png` - Payment gateway settings page (1200x900)
1. `screenshot-2.png` - Payment gateway settings page (1200x900)
1. `screenshot-3.png` - Payment gateway settings page (1200x900)
7. `screenshot-4.png` - WooCommerce Blocks checkout (1200x900)
8. `screenshot-5.png` - Transaction logs (1200x900)

### Plugin Icons (required)

- `icon-128x128.png` - Small icon (exactly 128x128)
- `icon-256x256.png` - Large icon (exactly 256x256)

### Plugin Banner (optional but recommended)

- `banner-772x250.png` - Low resolution banner
- `banner-1544x500.png` - High resolution banner (2x)

## Image Requirements

- Format: PNG or JPG
- Optimize images before upload
- Use high-quality screenshots
- Show actual plugin features
- Blur any sensitive information

## Notes

These files are used by WordPress.org for the plugin directory listing. They are not included in the plugin zip file.
